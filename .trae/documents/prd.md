## 1. 产品概述

城市共享单车流动数据分析平台是一个面向城市规划与运营人员的数据可视化工具。用户上传包含单车租借/还车时间与经纬度的 CSV 文件，系统通过 Pandas 聚合计算各区域当日的单车流入量、流出量（潮汐数据），并用 ECharts 在地图上以热力图与散点图呈现高频停放区域。

- 目标用户：共享单车运营团队、城市交通规划分析师
- 解决问题：快速识别单车“潮汐”聚集区，辅助车辆调度与站点规划
- 产品价值：将原始租借记录转化为直观的地图洞察，降低人工分析成本

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 分析员 | 无需注册（本地工具） | 上传 CSV、查看潮汐分析、切换可视化图层 |

### 2.2 功能模块

1. **数据上传页**：CSV 文件拖拽上传、字段校验、上传状态反馈
2. **潮汐分析仪表盘**：地图热力图、高频区域散点图、区域潮汐统计排行、全局指标卡片

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 数据上传页 | 上传卡片 | 支持拖拽/点击上传 CSV，显示文件名、大小、解析进度 |
| 数据上传页 | 字段说明 | 提示所需字段（租借时间、还车时间、租借经纬度、还车经纬度） |
| 数据上传页 | 示例数据 | 提供“加载示例数据”按钮，一键体验分析流程 |
| 潮汐分析仪表盘 | 全局指标卡 | 总租借次数、总还车次数、区域数、净流入最大区域 |
| 潮汐分析仪表盘 | 地图可视化 | 基于 ECharts 地图组件，叠加热力图与散点图层 |
| 潮汐分析仪表盘 | 区域排行 | 按净流量/活跃度排序的区域榜单，点击联动高亮地图 |
| 潮汐分析仪表盘 | 图层切换 | 切换热力图/散点图/合并视图 |

## 3. 核心流程

用户进入平台 → 上传 CSV（或加载示例数据）→ 前端将文件 POST 到后端 → 后端用 Pandas 读取并校验 → 按经纬度网格划分区域 → 计算各区域流入/流出/净流量 → 返回聚合结果 → 前端 ECharts 渲染地图热力图与散点图 → 用户切换图层、查看区域排行。

```mermaid
flowchart TD
    "A[用户进入平台]" --> "B{是否上传 CSV}"
    "B" -- "否" --> "C[加载示例数据]"
    "B" -- "是" --> "D[拖拽/选择 CSV 上传]"
    "C" --> "E[前端 POST 文件到 /api/analyze]"
    "D" --> "E"
    "E" --> "F[FastAPI 接收并 Pandas 读取]"
    "F" --> "G[字段校验与清洗]"
    "G --> "H[经纬度网格划分区域]"
    "H --> "I[计算流入/流出/净流量]"
    "I --> "J[聚合高频区域坐标]"
    "J --> "K[返回 JSON 结果]"
    "K --> "L[ECharts 渲染热力图与散点图]"
    "L --> "M[用户切换图层/查看排行]"
```

## 4. 界面设计

### 4.1 设计风格

- 主题：深色“指挥中心”风格，突出地图与数据光点
- 主色：深邃墨蓝底色 `#0A0E1A`，辅以青蓝色 `#2DD4BF`（流入）与琥珀色 `#F59E0B`（流出）双色对比表达潮汐方向
- 按钮：圆角胶囊按钮，主操作带青色辉光
- 字体：标题使用 `Space Grotesk`（数据感），正文使用 `Inter`；数字指标使用等宽 `JetBrains Mono` 突出可读性（注：按规范避开默认滥用字体，此处作为差异化数据展示）
- 布局：卡片式栅格 + 全宽地图主舞台，顶部状态栏

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 数据上传页 | 上传卡片 | 居中拖拽区、虚线边框、悬停辉光、文件信息列表 |
| 数据上传页 | 字段说明 | 半透明卡片、字段标签 chip |
| 仪表盘 | 顶部状态栏 | 文件名、记录数、刷新按钮 |
| 仪表盘 | 指标卡 | 4 列卡片、大号等宽数字、趋势小色块 |
| 仪表盘 | 地图主舞台 | 全宽 ECharts 地图、图层切换 Tab |
| 仪表盘 | 区域排行 | 右侧滚动列表、进度条、点击高亮 |

### 4.3 响应式

桌面优先设计（1280px+ 为最佳体验），平板端将仪表盘栅格降为双列，移动端单列堆叠并将地图高度自适应。地图组件与排行支持触控缩放。

### 4.4 数据可视化指引

- 热力图：基于区域网格点，颜色由冷（低活跃）到暖（高活跃），呈现停放密度
- 散点图：每个高频区域一个气泡，半径=活跃度，颜色=净流量方向（青=净流入，琥珀=净流出）
- 地图底图：使用 ECharts geo 组件，城市经纬度自适应居中缩放
